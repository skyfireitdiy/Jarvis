# 目录选择对话框组件规范

## 功能概述

将 App.vue 中的目录选择对话框拆分为独立的 Vue 组件 `DirectoryDialog.vue`。

主要功能：

- 目录树浏览
- 目录搜索过滤
- 目录导航（进入子目录、返回上级）
- 目录选择确认

## 接口定义

### Props

| 名称 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `visible` | Boolean | `false` | 对话框是否可见 |
| `initialPath` | String | `~` | 初始路径 |
| `nodeId` | String | `master` | 节点 ID |

### Events

| 事件名 | 参数 | 说明 |
|--------|------|------|
| `update:visible` | `Boolean` | 更新可见状态 |
| `select` | `path` | 选择目录 |
| `cancel` | - | 取消选择 |

## 内部状态

- `currentPath`: 当前浏览路径
- `directories`: 当前目录下的子目录列表
- `searchQuery`: 搜索关键词
- `isLoading`: 是否正在加载
- `selectedPath`: 选中的路径

## 功能行为

### 目录浏览

1. 显示当前路径
2. 列出当前目录下的所有子目录
3. 支持点击进入子目录
4. 支持返回上级目录

### 搜索过滤

1. 实时过滤目录列表
2. 支持模糊匹配
3. 搜索范围包括完整路径

### 路径选择

1. 点击目录选中
2. 支持手动输入路径
3. 确认按钮提交选择

### 键盘导航

1. 上下箭头选择目录
2. 回车键进入目录或确认选择
3. ESC 键取消操作

## 样式规范

- 类名前缀：`.dir-modal`
- 支持目录树状显示
- 高亮当前选中项
