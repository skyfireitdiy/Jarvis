# 在对话中固定长期有效的补充要求

## 使用场景

如果你希望某些要求在后续会话里持续生效，例如固定输出格式、统一语言风格、长期遵守的工作约束，可以把这些内容固定下来，避免每次重复输入。

## 操作步骤

1. 在当前对话输入框中先输入 `'<Pin>'`。
2. 紧接着把你希望长期保留的补充要求写在这个标记后面，例如固定输出格式、语言风格或长期约束。
3. 按 `Ctrl+J` 或 `Ctrl+]` 提交。
4. 提交后，系统会把 `'<Pin>'` 后面的内容保存为固定内容，后续继续执行其他任务时会持续生效。

## 你会看到的提示与反馈

- 固定成功后，系统会提示类似：`📌 已固定内容: ...`，并显示你固定内容的前一部分。
- 这类固定内容不会只影响当前一句输入，而是会持续参与后续对话。
- 之后你无需每轮都重复输入同一套补充要求。

## 注意事项

- `'<Pin>'` 后面应写稳定、长期适用的要求，不要塞入只对单次任务有效的临时说明。
- 如果你连续多次固定内容，系统会继续追加，而不是自动替换旧内容。
- 当固定内容越来越多时，也要警惕它对后续任务产生不必要约束。
