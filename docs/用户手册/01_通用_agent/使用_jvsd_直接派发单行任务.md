# 使用 jvsd 直接派发单行任务

## 使用场景

当你只想快速丢给系统一条简短任务，不想先进入完整交互流程时，可以使用 `jvsd` 直接派发单行任务。

## 操作步骤

1. 在命令行输入：`jvsd "你的任务"`。
2. 如果这不是一个现成文件路径，系统会自动在任务前补充内置规则入口和“原始需求”前缀。
3. 对于单行任务，系统会直接把内容交给 `jvs -n --task` 处理。
4. 提交后等待执行结果。

## 你会看到的提示与反馈

- `jvsd` 是 `jvs` 的便捷封装，适合快速派发。
- 如果传入内容为空，系统会直接报错并退出。
- 如果本机找不到 `jvs` 命令，系统会提示：`❌ 错误: 找不到 'jvs' 命令，请确保 jarvis 已正确安装`。

## 注意事项

- 单行模式适合短任务，不适合复杂长说明。
- 如果你的内容本身就是一个已存在的文件路径，系统会按任务文件而不是普通文本处理。
- 想输入多段说明时，建议改用多行方式。
