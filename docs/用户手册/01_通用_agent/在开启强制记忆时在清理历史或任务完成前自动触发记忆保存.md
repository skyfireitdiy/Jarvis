# 在开启强制记忆时在清理历史或任务完成前自动触发记忆保存

## 使用场景

当你担心重要信息会在清理历史或任务完成时被遗漏，希望系统在关键节点自动先做一次记忆保存尝试时，可以开启强制记忆模式。

## 操作步骤

1. 先进入已经开启强制记忆的会话或启动方式。
2. 正常执行任务。
3. 当系统准备清理历史，或任务即将完成时，会自动检查是否需要保存记忆。
4. 系统会静默尝试保存，不影响主流程继续进行。

## 你会看到的提示与反馈

- 强制记忆开启后，系统会在关键节点自动触发记忆保存逻辑。
- 如果保存成功，会看到：`✅ 已自动保存有价值的信息到记忆系统`
- 如果保存失败或没有值得保存的信息，也不会阻塞主流程。

## 注意事项

- 这项功能依赖当前入口已经开启强制记忆，不是默认任何会话都会自动执行。
- 这项功能是“关键节点兜底”，不是替代你所有主动记忆管理操作。
- 如果你不希望系统在结束阶段额外处理记忆，应关闭强制记忆功能。
