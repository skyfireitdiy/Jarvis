# 手动直接执行任务分析

## 使用场景

当一个任务已经做完，你希望系统立刻对这次工作做总结、保存记忆或沉淀经验时，可以在任务结束后按提示确认执行任务分析。

## 操作步骤

1. 先把当前任务做到一个明确的结束点，让系统完成本轮回答。
2. 任务结束后，留意系统是否弹出确认提示：`任务已完成，是否进行任务分析（保存记忆、生成方法论等）？`
3. 如果你希望马上开始分析，就确认继续；如果现在不想做分析，就选择跳过。
4. 如果继续，系统接着会询问：`您对本次任务的完成是否满意？`
5. 若你选择不满意，系统还会继续提示：`请提供您的反馈意见（可留空直接回车）`，你可以输入补充意见后提交。

## 你会看到的提示与反馈

- 任务分析不是在普通对话里随时输入一句话就立即触发，而是在任务完成后出现确认提示。
- 你会先看到：`任务已完成，是否进行任务分析（保存记忆、生成方法论等）？`
- 随后可能看到：
  - `您对本次任务的完成是否满意？`
  - `请提供您的反馈意见（可留空直接回车）`
- 如果你确认执行，系统会继续进行后续分析、记忆保存或经验沉淀流程。

## 注意事项

- 这项能力更适合任务已经告一段落时使用，不适合在中途频繁触发。
- 当前已确认的用户可见入口，是“任务完成后的确认提示”，而不是普通输入框里的独立内置命令。
- 如果你跳过这一步，本轮任务仍然结束，只是不继续做后续分析。
