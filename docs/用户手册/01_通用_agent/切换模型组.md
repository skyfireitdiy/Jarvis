# 切换模型组

## 使用场景

当你想在速度、成本和效果之间切换，例如先用较轻模型处理普通任务、再换到更强模型处理复杂任务时，可以切换模型组。

## 操作步骤

### 方式一：启动时指定

1. 启动通用代理时，在命令里使用 `-g` 或 `--llm-group` 指定模型组。
2. 提交命令后，系统会按你指定的模型组进入当前会话。

### 方式二：会话中切换

1. 在当前对话输入框中输入 `'<SwitchModel>'`。
2. 按 `Ctrl+J` 或 `Ctrl+]` 提交。
3. 系统会弹出模型组选择流程，你按提示选择目标模型组。
4. 选择成功后，继续执行当前任务。

## 你会看到的提示与反馈

- 启动阶段的欢迎信息会带上当前使用的平台和模型信息。
- 在会话内切换成功时，系统会提示：`✅ 模型组切换成功。`
- 如果你取消了切换或切换失败，系统会提示：`❌ 模型组切换失败或已取消。`
- 如果当前入口不支持模型组切换命令，你会先看到不可用提示，而不是直接进入选择。

## 注意事项

- 启动时指定适合在任务开始前就确定模型策略的场景。
- 会话中切换更适合任务做到一半、你临时想调整模型能力时使用。
- 切换失败时，优先检查当前会话是否支持该操作，以及目标模型组是否可用。
