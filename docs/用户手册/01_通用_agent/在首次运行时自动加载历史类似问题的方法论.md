# 在首次运行时自动加载历史类似问题的方法论

## 使用场景

当你第一次启动当前任务时，希望系统自动参考过去处理类似问题时沉淀下来的经验，而不是每次都从零开始摸索时，可以启用方法论能力。

## 操作步骤

1. 先进入已经开启方法论能力的启动方式或会话。
2. 正常输入当前任务，并按常规方式提交。
3. 在首次运行阶段，系统会根据当前任务内容自动查找历史类似问题的方法论。
4. 如果找到了合适经验，系统会自动把这些内容带入当前任务环境。

## 你会看到的提示与反馈

- 方法论加载发生在任务开始阶段，而不是任务结束后。
- 系统会根据当前任务内容自动检索相近经验，而不需要你手动先写检索词。
- 加载成功后，后续执行会自动参考这部分历史经验。

## 注意事项

- 这项能力依赖当前入口已经启用了方法论；如果没启用，就不会自动加载。
- 方法论适合参考，不代表必须逐条照搬。
- 如果当前任务和历史经验差异很大，方法论的参考价值可能有限。
