# 按会话名称保存并自动清理旧会话

## 使用场景

当你保存的会话越来越多时，系统会尽量给每个会话起一个便于识别的名称，并自动清理过旧记录，避免目录无限膨胀。

## 操作步骤

1. 先确保当前会话里已经有实际任务输入内容。
2. 如果当前入口支持会话保存，在对话输入框中输入 `'<SaveSession>'`，再按 `Ctrl+J` 或 `Ctrl+]` 提交。
3. 如果当前会话还没有名称，系统会根据你的原始输入生成一个简短中文名称。
4. 保存后的文件名会带上这个会话名称和时间戳。
5. 当同类会话保存太多时，系统会自动清理旧会话，只保留最近的一批。

## 你会看到的提示与反馈

- 首次生成名称时，系统会提示：`📝 生成会话名称: ...`
- 保存成功时，还会看到：`✅ 会话已成功保存。`
- 保存文件名中会包含会话名称、Agent 名称和时间戳，方便你区分不同会话。
- 系统会自动清理旧会话，默认只保留最近 10 个主会话及其辅助文件。

## 注意事项

- 会话名称来自原始任务内容的自动概括，不一定完全等于你的人工标题。
- 如果你是从旧会话恢复后再继续保存，系统会优先沿用原来的会话名称。
- 如果当前入口不支持 `'<SaveSession>'`，你需要换到支持会话保存的入口再操作。
