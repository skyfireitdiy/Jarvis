# 自动加载当前激活的规则内容

## 使用场景

当你希望代理在每次处理任务时自动遵循一组固定约束，而不是反复手动说明时，可以先激活规则，再让系统在启动后自动加载这些规则。

## 操作步骤

1. 先确认你需要的规则已经处于激活状态。
2. 如果不确定当前状态，先在对话输入框输入 `'<ListRule>'`，再按 `Ctrl+J` 或 `Ctrl+]` 查看哪些规则显示为 `✅ 已激活`。
3. 启动通用代理或继续当前会话。
4. 正常输入并提交任务，系统会自动把当前已激活规则带入本轮处理。

## 你会看到的提示与反馈

- 启动后不会要求你再次手动粘贴规则内容。
- 如果你随后查看规则状态，系统会把规则名称、内容预览、文件路径和是否已激活展示出来。
- 已激活的规则会以当前会话默认约束的方式生效。

## 注意事项

- 只有已经激活的规则会被自动带入当前任务环境。
- 如果你怀疑结果不符合预期，先检查是否激活了不适合当前任务的规则。
- 建议在正式执行复杂任务前先查看一次规则状态，避免带着错误规则开工。
