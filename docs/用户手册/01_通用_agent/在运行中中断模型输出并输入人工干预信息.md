# 在运行中中断模型输出并输入人工干预信息

## 使用场景

当你发现代理的回答方向跑偏了，或者它即将进行你不想要的动作时，可以在运行过程中打断它，并补充新的说明。

## 操作步骤

1. 当代理正在连续输出时，按中断快捷键打断当前输出。
2. 中断后，系统会立刻提示：`⚠ 检测到中断，请输入补充信息（Ctrl+J/Ctrl+]确认，直接回车跳过）`
3. 如果你想修正方向，就在这个输入框里直接写新的补充说明。
4. 写完后按 `Ctrl+J` 或 `Ctrl+]` 提交，系统会把这段内容作为人工干预继续带入后续流程。
5. 如果你不想补充，直接回车即可跳过。

## 你会看到的提示与反馈

- 中断后不会立即丢失当前任务，而是先给你一次人工补充机会。
- 你会先看到：`⚠ 检测到中断，请输入补充信息（Ctrl+J/Ctrl+]确认，直接回车跳过）`
- 你输入的内容会作为中断后的补充说明进入后续处理。
- 如果你直接回车，系统会视为跳过本次补充。

## 注意事项

- 中断适合用于纠偏，不适合替代完整任务输入。
- 补充信息应尽量直接说明你要修改什么方向、限制什么行为。
- 如果只是轻微补充，也尽量写得明确，避免代理继续误解。
