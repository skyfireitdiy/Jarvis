# 使用通用代理交互式执行任务

## 使用场景

适合你还需要一边执行、一边补充要求的任务，例如调试问题、逐步澄清需求、边看结果边修正方向。

## 操作步骤

1. 在终端中直接启动通用代理，不要带一次性任务参数，也不要开启非交互模式。
2. 启动后会进入多行输入界面，提示词为：`请输入你的任务（Ctrl+C 退出）`
3. 在输入区写下你的任务内容。
4. 输入完成后，按底部工具栏提示的提交快捷键提交。默认文案会显示为 `Ctrl+J`、`Ctrl+]` 等配置中的提交键。
5. 提交后，代理会立即开始处理你的任务。

## 你会看到的提示与反馈

- 输入框左侧会显示 `👤 >`
- 底部工具栏会显示快捷键提示，包括：
  - `@` 文件补全
  - 提交快捷键（如 `Ctrl+J/Ctrl+]`）
  - `Ctrl+O` 复制历史信息
  - `Alt+T` 进入终端
  - `Ctrl+X` 退出程序
  - `Ctrl+C/D` 取消
- 首次使用多行输入时，系统还会额外显示一次提交方式说明。

## 注意事项

- 如果你按 `Ctrl+C` 取消当前输入，本次任务不会提交。
- 如果你已经把任务写得非常完整，直接执行一次性任务通常更快。
- 交互式模式更适合需要多轮补充说明的场景。
