# 在任务结束时让系统自动判断是否保存长期记忆

## 使用场景

当你希望系统自己判断这次任务里有没有值得长期保存的经验、偏好或项目信息，而不是每次都手动决定是否记忆时，可以使用自动记忆判断。

## 操作步骤

1. 先使用支持记忆能力的会话或启动方式。
2. 正常执行并完成任务。
3. 在任务结束时，系统会自动回顾本次任务，判断是否有值得保存的信息。
4. 如果判断有价值，就会自动进入记忆保存流程；如果没有，就直接结束而不强行保存。

## 你会看到的提示与反馈

- 系统可能会给出与记忆相关的建议，例如在分析前建议保存关键信息。
- 如果自动保存成功，会显示：`✅ 已自动保存有价值的信息到记忆系统`
- 如果系统判断没有值得保存的信息，则不会强行保存。

## 注意事项

- 自动判断依赖当前入口具备记忆能力。
- 这项能力适合沉淀长期复用的信息，不适合把所有一次性内容都存进去。
- 如果你有明确想保留的重要内容，仍然建议手动保存，不要完全依赖自动判断。
