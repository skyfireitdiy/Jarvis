# 启动时加载并选择预定义任务

## 使用场景

如果你经常执行重复性工作，例如固定的检查流程、常见分析任务或标准化操作，可以在启动时先从预定义任务列表中选择，而不是每次重新编写完整任务。

## 操作步骤

1. 先准备好预定义任务列表，然后启动通用代理。
2. 启动后，系统会先列出可选任务的编号和名称。
3. 如果进入筛选式选择界面，你会看到提示：`选择一个任务编号 (ESC跳过) >`
4. 这时你可以直接选择某个编号；如果按 `ESC`，就会跳过预定义任务选择。
5. 如果当前环境没有进入筛选界面，系统会改为普通输入方式，并提示：`请选择一个任务编号（0 或者直接回车跳过预定义任务）：`
6. 选中后，系统会把对应任务内容作为本次执行的起点，并继续后续流程。

## 你会看到的提示与反馈

- 系统会先打印预定义任务列表，并显示编号。
- 你可能会看到两种选择方式中的一种：
  - `选择一个任务编号 (ESC跳过) >`
  - `请选择一个任务编号（0 或者直接回车跳过预定义任务）：`
- 如果你选择跳过，不会报错，而是继续进入普通任务输入或后续流程。
- 如果你选中了某个任务，系统会继续询问你是否要补充额外说明。

## 注意事项

- 只有已经准备好的预定义任务才会出现在列表里。
- 如果列表中没有合适的任务，直接跳过即可，不必强行选择。
- 任务名称只是入口，正式执行内容仍取决于你选择的具体任务和后续补充说明。
