# 为预定义任务追加补充信息

## 使用场景

当你已经从预定义任务列表里选中了一个任务，但这次执行还有额外背景、目标、限制或输入材料时，可以在正式执行前继续追加说明。

## 操作步骤

1. 先选择一个预定义任务。
2. 选中后，系统会询问：`需要为此任务添加补充信息吗？`
3. 如果你选择需要，系统会再打开多行输入框，并提示：`请输入补充信息`
4. 在输入框中写入本次执行特有的补充要求。
5. 写完后按 `Ctrl+J` 或 `Ctrl+]` 提交，系统会把预定义任务和补充信息合并后一起执行。

## 你会看到的提示与反馈

- 如果你选择不补充，系统会直接按原任务继续执行。
- 如果你选择补充，你会先看到：`请输入补充信息`
- 如果你补充了内容，这些补充说明会和预定义任务一起成为当前任务输入的一部分。
- 补充信息输入方式与普通多行输入一致，可使用相同的提交和取消按键。

## 注意事项

- 补充内容最好写本次任务独有的信息，不要把模板里已有内容重复一遍。
- 如果补充说明和原任务目标冲突，最终效果可能不稳定。
- 如果你在补充阶段取消输入，通常会回到不补充或结束当前追加流程的结果。
