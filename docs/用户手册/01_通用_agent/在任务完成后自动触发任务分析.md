# 在任务完成后自动触发任务分析

## 使用场景

当你希望任务做完后，系统自动进入收尾分析阶段，帮助整理结果、复盘过程，而不是只给出一句“完成了”时，可以开启自动任务分析。

## 操作步骤

1. 先进入已经启用任务分析能力的会话或启动方式。
2. 正常提交并完成任务。
3. 任务结束后，系统会出现确认提示：`任务已完成，是否进行任务分析（保存记忆、生成方法论等）？`
4. 如果你确认继续，系统会进入任务分析流程；如果你跳过，就直接结束这一轮分析。

## 你会看到的提示与反馈

- 任务完成后，系统不会只停在结束语，而是会先出现：`任务已完成，是否进行任务分析（保存记忆、生成方法论等）？`
- 如果你继续，后面还可能看到：`您对本次任务的完成是否满意？`
- 如果同时启用了任务总结，通常还会看到任务总结内容输出。
- 自动分析适合在任务结束后帮助你进一步沉淀结果、经验和后续行动。

## 注意事项

- 自动任务分析是否执行，取决于任务分析功能是否开启。
- 当前已确认的用户可见入口，是任务完成后的确认提示，而不是普通输入框中的独立命令。
- 如果你只想快速结束，不想进入额外分析，应检查当前配置是否启用了该能力。
