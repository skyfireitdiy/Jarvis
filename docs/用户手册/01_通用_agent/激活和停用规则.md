# 激活和停用规则

## 使用场景

当你希望某些规则只在当前任务或当前阶段生效，而不是一直保持启用时，可以先查看当前规则状态，再按需要切换启用状态。

## 操作步骤

1. 先在对话输入框输入 `'<ListRule>'`，然后按 `Ctrl+J` 或 `Ctrl+]` 查看当前规则列表。
2. 在列表里记下你要处理的规则名称，并先确认它当前是 `✅ 已激活` 还是 `🔴 未激活`。
3. 再使用当前入口支持的规则切换方式，按规则名称启用或停用对应规则。
4. 切换后，再次输入 `'<ListRule>'` 查看结果，确认状态是否已经变化。

## 你会看到的提示与反馈

- 启用成功后，再次查看时，该规则会从 `🔴 未激活` 变成 `✅ 已激活`。
- 停用成功后，再次查看时，该规则会从 `✅ 已激活` 变成 `🔴 未激活`。
- 激活后的规则会参与后续对话和任务执行。
- 停用后的规则只是不再影响后续流程，不会回头修改已经生成的内容。

## 注意事项

- 先查看状态，再切换规则，比盲目操作更稳妥。
- 如果当前入口不提供规则切换能力，你至少可以先用 `'<ListRule>'` 确认规则名和状态，再换到支持规则管理的入口继续操作。
- 激活过多规则可能让当前约束互相叠加，影响回答风格或执行方式。
