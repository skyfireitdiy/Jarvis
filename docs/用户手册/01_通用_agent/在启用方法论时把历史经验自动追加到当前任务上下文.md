# 在启用方法论时把历史经验自动追加到当前任务上下文

## 使用场景

当系统找到了历史类似问题的经验后，你希望这些经验能直接参与当前任务推理，而不是只放在外部备注里时，可以让方法论自动追加到当前上下文中。

## 操作步骤

1. 先进入已经启用方法论的启动方式或会话。
2. 正常输入当前任务，并按常规方式提交。
3. 系统会先处理你的当前输入，再自动检索对应的历史经验。
4. 找到合适经验后，系统会把这些内容直接追加到当前任务上下文里，作为后续执行参考。

## 你会看到的提示与反馈

- 追加后的经验会以类似“以下是历史类似问题的执行经验，可参考：”的形式进入当前上下文。
- 代理在后续回答和执行时，会自然参考这些经验。
- 你不需要手动复制粘贴过去的做法。

## 注意事项

- 这项能力依赖当前入口已经启用了方法论；如果没启用，就不会自动追加历史经验。
- 自动追加的是“参考经验”，不是强制规则。
- 如果你发现历史经验不适合当前问题，应及时在当前任务中明确修正方向。
