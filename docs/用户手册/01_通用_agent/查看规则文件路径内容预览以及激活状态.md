# 查看规则文件路径、内容预览以及激活状态

## 使用场景

当你不确定当前有哪些规则可用、这些规则来自哪里、是否已经激活时，可以先把规则列表调出来，再决定下一步加载哪一条。

## 操作步骤

1. 在当前对话输入框中直接输入 `'<ListRule>'`。
2. 按 `Ctrl+J` 或 `Ctrl+]` 提交这条指令。
3. 系统会立即列出当前可用规则，而不是把这段文字当成普通任务继续对话。
4. 查看列表中的规则名称、内容预览、文件路径和状态，再决定后续是否继续使用相关规则。

## 你会看到的提示与反馈

- 列表标题会显示为：`📋 所有可用规则`
- 表格中会包含以下几列：
  - `规则名称`
  - `内容预览`
  - `文件路径`
  - `状态`
- 状态列会显示为：
  - `✅ 已激活`
  - `🔴 未激活`
- 列表底部还会显示总数，例如：`总计: N 个规则`
- 内容预览只会展示前面一小段，方便你快速判断这条规则是不是你要找的内容。

## 注意事项

- 内容预览不是完整规则全文，只适合快速辨认用途。
- 文件路径过长时，界面里可能会显示为截断后的短路径。
- 这一步只是查看状态，不会自动帮你启用或停用规则。
