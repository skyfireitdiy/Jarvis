# 在 VSCode 插件中查看 Agent 列表与节点归属

## 使用场景

当你想在 VSCode 插件里快速判断有哪些 Agent、它们的运行状态，以及每个 Agent 运行在哪个节点上时，可以直接通过侧边栏 Agent 列表查看。

## 操作步骤

1. 先确保插件已经成功连接到 Jarvis Gateway。
2. 在侧边栏查看 Agent 列表。
3. 观察每个 Agent 的名称、状态、模型组、工作目录和节点标识。
4. 如需进一步交互，点击某个 Agent 打开对应聊天面板。

## 你会看到的提示与反馈

- 每个 Agent 会显示名称、运行状态和工作目录。
- 如果当前版本启用了 node 模式，列表中还会显示节点标识，例如 `🧭 master` 或对应子节点 ID。
- 新创建或新复制的 Agent 刷新后会出现在列表前部。

## 注意事项

- 节点信息依赖后端返回的 `node_id`。
- 即使某个 Agent 运行在子节点，插件端仍然只需要连接主节点。
