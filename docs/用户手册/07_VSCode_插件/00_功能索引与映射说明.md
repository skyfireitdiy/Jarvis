# VSCode 插件功能索引

## 使用场景

当你希望直接在 VSCode 侧边栏中连接 Jarvis Gateway、查看 Agent 列表、创建或复制 Agent，并在编辑器工作流中持续使用 Jarvis 时，可以先从这份索引快速找到对应文档。

## 操作步骤

1. 先确认你已经安装并启用了 Jarvis VSCode 插件。
2. 再确认至少有一个可连接的 Jarvis Gateway 实例。
3. 根据你当前的目标，在下方索引中找到对应文档并按步骤操作。

## 你会看到的提示与反馈

- 本目录主要覆盖 VSCode 插件中的连接、Agent 管理、节点选择与节点展示能力。
- 如果你使用的是 node 模式，插件会在连接主节点后加载可用节点列表。
- 大部分文档采用“使用场景 / 操作步骤 / 你会看到的提示与反馈 / 注意事项”的固定结构。

## 注意事项

- VSCode 插件通常连接主节点，而不是直接连接子节点。
- 如果主节点尚未发现任何子节点，插件中的目标节点下拉框可能只显示默认主节点。
