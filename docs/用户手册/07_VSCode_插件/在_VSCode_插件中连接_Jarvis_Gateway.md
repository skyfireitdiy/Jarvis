# 在 VSCode 插件中连接 Jarvis Gateway

## 使用场景

当你希望不离开 VSCode，就能直接访问 Jarvis 的 Agent 列表与对话能力时，可以在 VSCode 插件侧边栏中连接 Jarvis Gateway。

## 操作步骤

1. 在 VSCode 中打开 Jarvis 插件侧边栏。
2. 在连接区域输入 Gateway 地址，例如 `127.0.0.1:8000`。
3. 如果当前 Gateway 开启了密码认证，再输入对应密码。
4. 点击“连接”。
5. 连接成功后，侧边栏会自动切换到 Agent 列表视图。

## 你会看到的提示与反馈

- 连接成功后，插件会显示当前可用 Agent 列表。
- 如果当前主节点开启了 node 模式，插件还会自动加载可用节点列表，供后续创建 Agent 使用。
- 如果登录失败或地址不可达，侧边栏会显示错误提示。

## 注意事项

- VSCode 插件通常连接主节点，而不是直接连接子节点。
- 如果密码错误、token 失效或主节点不可达，插件需要重新登录。
