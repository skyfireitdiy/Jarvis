# 启用智能增强能力

## 使用场景

如果你希望系统在歧义处理、主动交互、对话管理等方面更积极，可以配置 `enable_autonomous`。

## 操作步骤

1. 打开 Jarvis 全局配置文件：`~/.jarvis/config.yaml`。
2. 添加或修改以下配置：

   ```yaml
   enable_autonomous: true
   ```

3. 保存配置文件。
4. 重新启动相关会话。
5. 在一个含糊、需要澄清或需要系统主动协助管理对话的任务中，观察系统行为是否更积极。

## 你会看到的提示与反馈

- 开启后，系统在情绪识别、歧义检测、对话管理、主动交互方面会更积极。
- 如果关闭或不配置，系统会保持默认较保守的行为。
- 这类变化通常体现在对话过程本身，而不是单独弹出“智能增强已启用”的提示。

## 注意事项

- 配置项名称是 `enable_autonomous`。
- 默认值是 `false`。
- 这项能力更适合需要更主动协助的场景；如果你追求完全简洁、低干预交互，可以保持关闭。
