# 查看启动时可用资源统计

## 使用场景

如果你想快速确认当前环境是否已经正确加载方法论、工具和记忆，可以关注 Agent 启动时显示的资源统计面板。

## 操作步骤

1. 启动对应 Agent。
2. 在启动完成后，观察欢迎信息和资源统计面板。
3. 重点关注以下内容：
   - 当前平台与模型
   - 本地方法论数量
   - 可用工具数量
   - 全局记忆数量
   - 项目记忆数量（如果存在）
   - 短期记忆数量（如果存在）
4. 如果你刚修改过方法论目录、工具目录或记忆相关配置，可以用这里的数量变化做一次快速验证。

## 你会看到的提示与反馈

- 启动时会显示资源概览，帮助你了解当前环境已加载的资源规模。
- 如果项目记忆或短期记忆存在，它们会额外显示在统计信息里。
- 这是一个非常适合做“配置是否生效”快速核对的入口。

## 注意事项

- 这是启动时的自动反馈，不是需要额外配置某个键才能看到的独立功能入口。
- 如果你发现数量与预期不符，应反查对应目录、配置来源或过滤规则是否正确。
