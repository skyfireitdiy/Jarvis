# 开启会话历史保存

## 使用场景

如果你希望保留模型会话历史，方便后续追溯、恢复或延续上下文，可以配置 `save_session_history`。

## 操作步骤

1. 打开 Jarvis 全局配置文件：`~/.jarvis/config.yaml`。
2. 添加或修改以下配置：

   ```yaml
   save_session_history: true
   ```

3. 保存配置文件。
4. 重新启动相关会话。
5. 进行一次对话或任务执行。
6. 之后检查会话历史是否已经被保存，并在需要时用于后续恢复或追溯。

## 你会看到的提示与反馈

- 开启后，会话过程会被保存下来，便于后续查看或恢复。
- 如果未开启，默认不会保存会话历史。
- 相关效果通常体现在后续可恢复、可追溯，而不是即时弹出单独提示。

## 注意事项

- 配置项名称是 `save_session_history`。
- 默认值是 `false`。
- 如果你的会话涉及敏感信息，开启前应先确认本地保存策略是否符合你的安全要求。
