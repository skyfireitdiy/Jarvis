# 列出所有可用工具

## 使用场景

当你想先了解 Jarvis 当前提供了哪些工具、每个工具大致做什么，以及是否需要进一步查看参数说明时，可以先使用工具系统的列表命令。

## 操作步骤

1. 在终端中进入可使用 Jarvis 命令的环境。
2. 运行工具系统的列表命令，查看当前所有可用工具。
3. 如果你只想快速浏览工具名称和描述，使用默认输出即可。
4. 如果你希望程序或脚本进一步处理结果，可以改用 JSON 输出模式。
5. 如果你还想连同每个工具的参数结构一起查看，可以开启详细模式。
6. 如有需要，可以把默认输出和详细输出结合使用，先找工具，再决定是否查看单个工具详情。

## 你会看到的提示与反馈

- 默认模式下，终端会显示“可用工具列表”。
- 每个工具通常会显示：

  - 工具名称
  - 工具描述

- 开启详细模式后，还会看到每个工具的参数结构。
- 使用 JSON 模式时，输出会变成结构化 JSON：

  - 普通 JSON 模式只包含名称和描述
  - 详细 JSON 模式会包含更完整的工具信息

## 注意事项

- 这是命令行操作，不是 Web 页面里的功能。
- 默认模式更适合人工阅读；JSON 模式更适合脚本处理或复制给其他系统。
- 工具数量和内容取决于当前已注册的工具集合，不同环境下可能略有差异。
- 如果你已经知道某个工具名称，下一步通常可以直接查看该工具详情。
