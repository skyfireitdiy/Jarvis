# 列出所有可用工具

## 使用场景

当你想先了解 Jarvis 当前提供了哪些工具、每个工具大致做什么，以及是否需要进一步查看参数说明时，可以先使用工具系统的列表命令。

## 操作步骤

1. 在终端中进入可使用 Jarvis 命令的环境。
2. 如果你只想快速浏览工具名称和描述，运行：`jt list`。
3. 如果你希望程序或脚本进一步处理结果，运行：`jt list --json`。
4. 如果你还想连同每个工具的参数结构一起查看，运行：`jt list --detailed`。
5. 如果你既想看详细字段，又想交给程序处理，可以组合使用：`jt list --detailed --json`。
6. 查看结果后，可以先记下目标工具名称，再继续使用“查看单个工具详情”命令确认具体参数。

## 你会看到的提示与反馈

- 运行 `jt list` 时，终端会显示“可用工具列表”。
- 每个工具通常会显示：

  - 工具名称
  - 工具描述

- 运行 `jt list --detailed` 后，还会看到每个工具的参数结构。
- 运行 `jt list --json` 时，输出会变成结构化 JSON：

  - 普通 JSON 模式只包含名称和描述
  - `jt list --detailed --json` 会包含更完整的工具信息

## 注意事项

- 这是命令行操作，不是 Web 页面里的功能。
- 默认模式更适合人工阅读；`--json` 更适合脚本处理或复制给其他系统。
- `--detailed` 会让输出更长，适合在你准备进一步调用工具前确认参数结构。
- 工具数量和内容取决于当前已注册的工具集合，不同环境下可能略有差异。
- 如果你已经知道某个工具名称，下一步通常可以直接查看该工具详情。
