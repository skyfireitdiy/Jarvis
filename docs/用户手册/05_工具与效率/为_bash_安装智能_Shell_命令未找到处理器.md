# 为 bash 安装智能 Shell 命令未找到处理器

## 使用场景

当你使用 bash，希望在输入一段较长但不存在的命令后，由系统生成更合理的建议命令，并允许你先编辑再决定是否执行时，可以为 bash 安装智能 Shell 处理器。

## 操作步骤

1. 确认你正在使用或准备使用 bash shell。
2. 在终端中运行 JSS 的安装命令，并把 shell 类型指定为 bash。
3. 如果本机尚未存在 `~/.bashrc`，系统会自动创建该文件。
4. 安装完成后，按提示重新加载 bash 配置。
5. 之后，当你在 bash 中输入一段较长但不存在的命令时，系统会先生成建议命令。
6. 在交互式 bash 中，这条建议会预填到命令输入区域，你可以直接回车执行，也可以先编辑后再执行。

## 你会看到的提示与反馈

- 安装成功后，终端会提示 bash completion 已安装。
- 同时会提示你执行：`source ~/.bashrc`。
- 如果之前已经安装过，再次执行时会直接提示已安装。
- 在交互式 bash 中触发后，建议命令会被预填，你可以修改再执行。
- 在非交互模式下，系统只会打印建议命令，而不会进入可编辑输入状态。

## 注意事项

- bash 和 fish、zsh 的交互方式不同；bash 更偏向“预填后可编辑”。
- 如果不重新加载 `~/.bashrc`，当前终端会话中可能不会立即生效。
- 处理器只对较长的未识别输入生效，过短输入不会触发。
- 触发后是否真正执行，取决于你在交互式环境中是否确认并回车。
