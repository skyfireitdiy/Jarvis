# 工具与效率功能索引

## 使用场景

当你想快速判断“工具与效率”目录中的哪篇文档最适合当前问题，例如先查看有哪些工具、直接调用某个工具、整理长期记忆、管理方法论，或者给 Shell 增加自然语言命令辅助时，可以先看这份索引。

## 操作步骤

1. 先根据你当前要完成的事情，在下面的分类中找到最接近的场景。
2. 再打开对应文档，按文档中的步骤操作。
3. 如果多个草案功能点共用同一篇文档，说明这些能力在真实使用中属于同一条连续操作链，正式手册采用了合并覆盖的写法。
4. 如果某个草案能力没有单独成文，不一定代表缺失，也可能已经合并进同主题文档，或因为用户价值较低而被有意识收敛。

## 你会看到的提示与反馈

- 本目录中的文档主要覆盖以下几类能力：

  - 工具系统命令行
  - Git 提交效率
  - 长期记忆整理与迁移
  - 方法论导入、导出与提取
  - 规则索引查看
  - 智能 Shell 安装与命令生成

- 大部分文档都按“使用场景 / 操作步骤 / 你会看到的提示与反馈 / 注意事项”的固定结构编写。
- 少数草案点会合并到同一篇文档中，避免把连续命令链拆得过碎。

## 注意事项

- 这一目录主要是命令行能力，不是 Web 页面功能。
- 如果你发现草案中的某个条目没有独立文档，应优先判断是否已经被现有文档实质覆盖，而不是只看标题是否一一对应。
- 对用户价值不高、容易误导或更像内部机制说明的内容，正式手册会有意识收敛，不会为了凑条目而强行成文。
