# 查看全部可用规则索引

## 使用场景

当你想快速了解当前系统中有哪些可用规则，以及每条规则的大致说明时，可以使用规则索引查看命令。

## 操作步骤

1. 在终端中运行规则索引查看命令。
2. 如果你想要更适合人读的结果，直接使用默认模式即可。
3. 如果你需要结构化结果，可以改用 JSON 输出模式。
4. 如果你希望拿到未经美化处理的原始内容，可以改用 raw 模式。
5. 查看输出后，再决定是否继续在其他命令中加载或使用相关规则。

## 你会看到的提示与反馈

- 默认模式下，系统会以较易阅读的格式显示全部可用规则及其说明。
- JSON 模式下，输出会更适合程序处理或保存。
- raw 模式下，系统会直接打印原始内容，而不会做额外美化。
- 输出中通常会包含多类规则来源的索引信息，便于你快速总览。

## 注意事项

- 这个命令的目标是“查看规则索引”，不是直接加载规则正文。
- 默认模式适合人工阅读；JSON 和 raw 更适合自动化处理、复制或比对。
- 如果你只需要快速确认有哪些规则，默认模式通常已经足够。
- 规则索引内容会随当前可用规则集合变化，不同环境下可能不完全一致。
