# 导出记忆到 JSON 文件

## 使用场景

当你希望备份长期记忆、迁移到其他环境，或者只导出某一类或某些标签相关的记忆时，可以把记忆导出为 JSON 文件。

## 操作步骤

1. 在终端中运行记忆导出命令。
2. 指定导出文件路径。
3. 根据需要选择导出范围：

   - 同时导出项目长期记忆和全局长期记忆
   - 只导出其中一种

4. 如果你只想导出某些主题内容，可以再加上标签过滤条件。
5. 提交命令后，等待系统将符合条件的记忆写入 JSON 文件。
6. 导出完成后，检查目标 JSON 文件是否已生成。

## 你会看到的提示与反馈

- 导出成功后，系统会提示成功写入，并显示导出的数量或目标文件位置。
- 导出的 JSON 中通常包含：

  - 记忆内容
  - 标签
  - 记忆类型
  - 其他基础字段

- 如果你指定了标签过滤，最终导出的结果只会保留匹配这些标签的记忆。
- 如果参数不合法或目标路径不可写，系统会直接提示失败原因。

## 注意事项

- 导出的是 JSON 文件，适合做备份、迁移和批量处理。
- 标签过滤更适合做专题导出，但如果标签写得过窄，可能导出结果为空。
- 导出不会删除原始记忆，它只是把当前数据复制到文件中。
- 若你计划后续导入其他环境，建议保留原始导出文件，不要手动破坏其 JSON 结构。
