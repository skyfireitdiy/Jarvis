# 为 zsh 安装智能 Shell 命令未找到处理器

## 使用场景

当你使用 zsh，希望在输入一段较长但不存在的命令后，由系统先给出建议命令，并允许你在编辑器里修改后再执行时，可以为 zsh 安装智能 Shell 处理器。

## 操作步骤

1. 确认你正在使用或准备使用 zsh shell。
2. 在终端中输入：`jss install --shell zsh`。
3. 如果本机尚未存在 `~/.zshrc`，系统会自动创建该文件。
4. 安装完成后，按提示重新加载 zsh 配置。
5. 之后，当你在 zsh 中输入一段较长但不存在的命令时，系统会先生成建议命令。
6. 在交互式 zsh 中，系统会调用你的编辑器打开建议命令，你可以修改后保存退出，再继续执行。

## 你会看到的提示与反馈

- 安装成功后，终端会提示 zsh completion 已安装。
- 同时会提示你执行：`source ~/.zshrc`。
- 如果已经安装过，再次安装时会直接提示已安装。
- 在交互式 zsh 中触发后，系统会打开当前编辑器，让你先修改建议命令。
- 在非交互模式下，系统只会打印建议命令，而不会打开编辑器。

## 注意事项

- zsh 的交互体验和 bash、fish 不同，它更偏向“在编辑器中修改后再执行”。
- 当前使用哪个编辑器，取决于你的 `VISUAL` 或 `EDITOR` 环境设置；若未设置，会回退到默认编辑器。
- 不重新加载 `~/.zshrc` 时，当前终端会话中可能不会立刻生效。
- 处理器只针对较长的未识别输入触发，过短输入不会介入。
