# 为 fish 安装智能 Shell 命令未找到处理器

## 使用场景

当你使用 fish 作为日常 Shell，希望在输入一段较长但不存在的命令后，由系统自动把它转成建议命令并替换到命令行中时，可以为 fish 安装智能 Shell 处理器。

## 操作步骤

1. 确认你正在使用或准备使用 fish shell。
2. 在终端中输入：`jss install --shell fish`。
3. 如果本机尚未存在 fish 的配置文件，系统会自动创建所需目录和配置文件。
4. 安装完成后，按提示重新加载 fish 配置。
5. 之后，当你在 fish 中输入一个较长但不存在的命令时，系统会尝试把这段内容转换为建议命令并回填到命令行。

## 你会看到的提示与反馈

- 安装成功后，终端会提示 fish completion 已安装。
- 同时会提示你执行：`source ~/.config/fish/config.fish`。
- 如果之前已经安装过，再次安装时会直接提示已安装，而不会重复写入。
- 生效后，当输入较长但无法识别的命令时，系统会尝试给出建议命令并替换当前命令行内容。

## 注意事项

- 这里安装的是 fish 专用处理器，不适用于 bash 或 zsh。
- 安装完成后，如果不重新加载配置，当前 shell 会话里可能不会立刻生效。
- 处理器只会在命令足够长时介入，过短输入不会触发。
- 这项能力的目标是“给出并回填建议命令”，不是保证自动执行任何操作。
