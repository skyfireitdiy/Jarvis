# 调整侧边栏和浮动窗口布局

## 使用场景

当你在 Web 界面里同时使用 Agent 列表、终端面板和在线编辑器，希望把界面排布调整得更适合自己的屏幕大小和当前任务时，可以使用侧边栏宽度调整、浮动窗口拖动、缩放和最大化功能。

## 操作步骤

### 调整 Agent 侧边栏宽度

1. 在桌面端打开左侧 Agent 侧边栏。
2. 把鼠标移动到侧边栏右边缘。
3. 按住边缘拖动，即可调整侧边栏宽度。
4. 松开鼠标后，新的宽度会被记住。

### 调整终端面板

1. 打开独立终端面板。
2. 在桌面端，按住面板顶部标题栏可拖动位置。
3. 拖动面板边缘或角上的缩放区域，可调整终端面板大小。
4. 点击面板右上角最大化按钮，可在“最大化”和“还原”之间切换。

### 调整编辑器面板

1. 打开在线编辑器。
2. 在桌面端，按住编辑器顶部标题栏可拖动位置。
3. 拖动边缘或角上的缩放区域，可调整编辑器大小。
4. 点击编辑器右上角最大化按钮，可在“最大化”和“还原”之间切换。

### 打开或关闭相关面板

1. 如需显示或隐藏 Agent 侧边栏、终端面板或编辑器面板，优先使用页面上对应的按钮。
2. 打开终端或编辑器后，可以继续拖动、缩放或最大化窗口。
3. 如需关闭弹层或临时面板，可结合页面上的关闭按钮与 Esc 键操作。

## 你会看到的提示与反馈

- 终端和编辑器面板右上角都会显示最大化或还原按钮。
- 当前被你操作的浮动窗口会提升到更靠前的位置，便于继续使用。
- 在桌面端调整完成后，再次打开页面时，系统通常会沿用你上次的窗口位置、尺寸和侧边栏宽度。
- 在移动端下，编辑器更接近全屏展示，侧边栏通常会占满屏幕宽度，而不是保留桌面端那种自由拖动体验。
- 按 Esc 时，如果当前有设置窗口、目录选择框或会话恢复窗口，页面会优先关闭这些弹层。

## 注意事项

- 侧边栏拖拽调宽主要是桌面端能力，移动端通常不会提供同样的宽度调整方式。
- 终端和编辑器的拖动、缩放同样主要面向桌面端；移动端更偏向覆盖式展示。
- 当前文档仅保留已确认的按钮、拖拽、缩放、最大化与 Esc 行为；如果你所在环境还有额外快捷键，请以页面实际表现为准。
- 调整布局只是改变界面展示，不会修改 Agent、文件或终端本身的内容。
