# 使用补全面板插入常用内容

## 使用场景

当你在 Web 界面的多行输入框中不想手动记住全部可用项，想快速插入常见内容、规则引用或文件路径时，可以使用补全面板。

## 操作步骤

1. 先切换到一个可用的 Agent。
2. 把光标放到多行输入框中。
3. 触发补全面板有两种方式：

   - 直接在输入框中输入 @
   - 点击输入区右侧的 @ 按钮

4. 页面会弹出标题为“插入补全”的面板。
5. 如果补全项很多，可以在“搜索补全...”输入框中输入关键词筛选结果。
6. 选择补全项有两种方式：

   - 用上下方向键移动选中项，再按 Enter
   - 直接用鼠标点击目标项

7. 选中后，系统会把该补全内容插入到当前光标位置。
8. 如果你按 Esc 或点击右上角关闭按钮，补全面板会关闭，并在原来的触发位置保留一个 @，方便你继续手动输入。

## 你会看到的提示与反馈

- 打开面板后，会看到标题“插入补全”。
- 搜索框占位提示为“搜索补全...”。
- 如果没有匹配结果，列表中会显示“没有找到匹配的补全”。
- 当你输入搜索词时，补全面板除了筛选常规补全项外，还可能额外显示文件相关结果，便于你快速插入路径。
- 成功插入后，补全面板会自动关闭，光标会回到输入框中。
- 如果你用 Esc 或右上角关闭按钮退出，输入框里会保留一个 @，而不是完全不留内容地关闭。

## 注意事项

- 补全面板只作用于多行输入框，不是所有输入场景都会弹出。
- 你可以用 Esc 或右上角关闭按钮关闭面板；关闭时系统会在触发位置保留一个 @。
- 文件相关补全依赖当前 Agent 的工作目录，因此应先确认你已经切换到正确的 Agent。
- 移动端下补全面板不一定会像桌面端那样自动聚焦搜索框，但整体操作流程一致。
