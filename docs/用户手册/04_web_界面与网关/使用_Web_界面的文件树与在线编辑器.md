# 使用 Web 界面的文件树与在线编辑器

## 使用场景

当你想在 Jarvis 的网页界面里直接浏览 Agent 工作目录、打开文件、查看内容并进行在线编辑时，可以使用文件树和浮动编辑器，而不必切回本地终端或外部编辑器。

## 操作步骤

### 浏览文件树

1. 先选择一个已连接的 Agent。
2. 在左侧 Agent 列表中，点击该 Agent 的工作目录文字。
3. 首次展开时，如果还没有加载文件树，点击“加载文件树”。
4. 在文件树中点击目录，可继续展开或收起下级内容。
5. 点击文件时，会直接在编辑器中打开这个文件。

### 打开并查看文件

1. 在文件树中点击一个文件。
2. 页面会自动打开“编辑器”浮动面板。
3. 如果已经打开过这个文件，会直接切换到已有标签页。
4. 如果这是新文件，会新建一个标签页，并显示当前文件路径。

### 切换只读或可编辑模式

1. 打开编辑器后，查看顶部工具栏中的锁定状态。
2. 默认情况下，编辑器是只读模式。
3. 点击切换按钮后，可在“只读”和“可编辑”之间切换。

### 保存文件

1. 先切换到可编辑模式。
2. 修改文件内容。
3. 点击保存按钮。
4. 保存成功后，工具栏状态会更新，并显示成功提示。

### 关闭标签或关闭编辑器

1. 如需关闭单个文件标签，点击标签上的关闭按钮。
2. 如需关闭整个编辑器面板，点击编辑器面板右上角关闭按钮。
3. 如果当前存在未保存修改，系统会先弹出确认提示。

## 你会看到的提示与反馈

- 文件树中，目录和文件会以不同图标显示。
- 编辑器未打开文件时，会显示“点击文件树中的文件打开代码编辑器”。
- 编辑器工具栏会显示：

  - 加载中
  - 错误信息
  - 已保存
  - 未保存修改

- 保存成功后，会看到“文件已保存”的提示。
- 如果文件在外部被修改：

  - 若当前没有未保存修改，页面会自动刷新最新内容
  - 若当前有未保存修改，页面会提示存在外部变更，避免静默覆盖

- 如果关闭未保存标签或关闭整个编辑器，页面会先要求确认。

## 注意事项

- 编辑器默认是只读模式，想修改文件前需要手动切换到可编辑模式。
- 不是所有文件都支持打开和编辑，网页编辑器只适合文本文件。
- 文件过大、不是普通文本文件或不满足读取条件时，网页中可能无法正常打开。
- 编辑器支持多标签、窗口拖动、缩放、最大化和还原；在移动端会更偏向全屏式展示。
