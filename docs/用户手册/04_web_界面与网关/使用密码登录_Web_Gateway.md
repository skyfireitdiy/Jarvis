# 使用密码登录 Web Gateway

## 使用场景

当 Web Gateway 启用了密码认证时，你需要先在网页连接前输入密码，换取访问凭证，之后才能正常访问 Agent 列表、文件、终端和其他 Web 功能。

## 操作步骤

1. 先确保启动 Web Gateway 时已经设置了网关密码。
2. 在浏览器中打开 Web 界面，进入“连接到 Jarvis”窗口。
3. 在“密码”输入框中填写正确密码。
4. 在“网关地址”输入框中填写 Web Gateway 地址。
5. 点击“连接”。
6. 页面会先使用你输入的密码进行登录，再继续建立后续连接。
7. 登录成功后，页面会进入已连接状态，你可以继续使用其他网页功能。

## 你会看到的提示与反馈

- 如果登录成功，页面会继续完成连接流程，并自动关闭连接弹窗。
- 登录成功后，密码输入框中的内容会被自动清空。
- 如果密码错误或登录失败，连接窗口中会显示失败原因，而不会进入已连接状态。
- 登录成功后，后续接口和连接会自动带上当前访问凭证。

## 注意事项

- 这个密码不是每次发送操作时重复输入，而是在连接前先登录一次。
- 登录成功后，访问凭证只保存在当前页面运行期间；如果你刷新页面或重新打开页面，通常需要重新登录。
- 如果网关没有启用密码认证，这一步可以跳过，密码输入框也可以留空。
- 如果你已经连接成功，但后续凭证失效或网关重启，页面可能重新回到连接弹窗，需要再次登录。
