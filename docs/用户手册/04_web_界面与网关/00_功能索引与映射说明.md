# Web 界面与网关功能索引

## 使用场景

当你通过浏览器使用 Jarvis 时，可以先从这份索引判断当前功能应该看哪篇文档。它适合在你刚接触 Web 界面、准备开始连接网关，或者已经连接成功但不知道某个按钮和面板该怎么用时使用。

## 操作步骤

1. 先根据你当前要完成的事情，在下面的分类中找到最接近的场景。
2. 再打开对应文档，按文档中的步骤操作。
3. 如果你使用的是手机或窄屏窗口，优先留意文档中的移动端说明，因为侧边栏、终端面板和编辑器的展示方式会与桌面端不同。
4. 如果某个功能需要你已经连接到 Web Gateway，先完成以下基础步骤：

   - 启动 Web Gateway 服务
   - 在浏览器中连接 Jarvis Gateway
   - 如有需要，再使用密码登录或调整连接锁定模式

## 你会看到的提示与反馈

- 本目录中的文档主要覆盖以下几类能力：

  - 启动、连接与重新连接
  - Agent 管理与会话恢复
  - 输入、补全与人工介入
  - 文件树、编辑器与终端
  - 历史记录、设置与界面体验

- 大部分文档都按“使用场景 / 操作步骤 / 你会看到的提示与反馈 / 注意事项”的固定结构编写。
- 如果某项能力是自动发生的，文档中会明确说明你不需要额外输入命令。

## 注意事项

- Web 界面中的很多能力都依赖你已经成功连接到 Gateway；如果当前顶部仍显示“未连接”，请先处理连接问题。
- 某些能力只在桌面端更完整，例如侧边栏拖拽调宽、终端与编辑器的自由缩放；移动端通常会更偏向全屏或覆盖层展示。
- 如果你看到的界面按钮与文档描述不一致，优先确认自己是否处在不同设备宽度、不同 Agent 状态，或当前 Agent 是否已经完成。
