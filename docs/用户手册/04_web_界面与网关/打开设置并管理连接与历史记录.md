# 打开设置并管理连接与历史记录

## 使用场景

当你已经进入 Web 界面，想修改网关地址、重新输入密码、切换连接锁定开关，或者查看并清除当前浏览器里保存的历史记录时，可以使用设置弹窗。

## 操作步骤

1. 在 Web 界面中打开“设置”弹窗。
2. 在弹窗里按需查看或修改以下内容：

   - “密码”输入框
   - “网关地址”输入框
   - “锁定连接（拒绝新连接）”开关
   - 历史消息数量和存储空间信息

3. 如果你只是想调整连接参数，可以先修改对应内容。
4. 如果你要清空当前保存的历史记录，点击“清除历史记录”。
5. 如果你修改完后希望按新配置重新建立连接，点击“重新连接”。
6. 如果暂时不处理，点击“取消”或右上角关闭按钮离开设置弹窗。

## 你会看到的提示与反馈

- 设置弹窗标题显示为“设置”。
- 连接相关输入项会显示“密码”和“网关地址”。
- 连接锁定开关旁会显示“锁定连接（拒绝新连接）”以及说明文字。
- 历史消息区域会显示“历史消息数量”和“存储空间”。
- 当本地没有历史记录时，“清除历史记录”按钮会处于不可用状态。
- 底部操作区会显示“取消”和“重新连接”按钮。

## 注意事项

- “重新连接”会按你当前在设置弹窗中填写的连接信息重新建立连接，因此修改后应确认地址和密码是否正确。
- 连接锁定开关控制的是是否拒绝新的连接请求；如果你不确定当前环境是否需要独占连接，先保持现状更稳妥。
- 历史记录统计和清理面向当前浏览器中保存的数据；更换浏览器或设备后，你看到的历史数量可能不同。
