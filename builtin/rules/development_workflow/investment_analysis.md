# 投资分析框架（价值投资通用版）

## 简介

整体分为**7大模块**，层层递进：
宏观环境 → 行业赛道 → 公司基本面 → 财务数据 → 估值定价 → 风险排查 → 操作策略

适合个股深度分析、选股复盘、研报撰写。

## 数据获取方式

分析所需数据统一通过 **search_web** 工具搜索互联网获取：

- 宏观经济数据、货币政策、利率走势
- 行业研报、政策动态、竞争格局等定性信息
- 公司财务数据、估值指标、历史分位
- 管理层背景、股权结构、公司公告
- 可比公司对标、行业均值
- 风险事件、监管政策、新闻舆情
- ESG评级、社会责任报告

**搜索建议**：

- 宏观数据搜索：`"国家名" GDP增速 CPI 利率 货币政策`
- 财务数据搜索：`"公司名" 年报 营收 ROE 毛利率`
- 估值数据搜索：`"公司名" PE PB 估值分位`
- 行业分析搜索：`"行业名" 市场规模 增速 竞争格局`
- 风险信息搜索：`"公司名" 风险 商誉减值 诉讼`
- ESG信息搜索：`"公司名" ESG评级 社会责任报告`

## 你必须遵守的原则

### 1. 分析顺序原则

- **必须**：按7大模块顺序逐层分析，不得跳过
- **必须**：每个模块得出结论后再进入下一模块
- **禁止**：先看估值再倒推逻辑（避免确认偏差）

### 2. 数据驱动原则

- **必须**：每个判断必须有数据支撑
- **必须**：区分事实与推测，推测需标注
- **禁止**：无数据支撑的主观臆断

### 3. 风险优先原则

- **必须**：风险排查模块不得省略
- **必须**：列出至少3个风险因素
- **禁止**：只讲利好不提风险

### 4. 安全边际原则

- **必须**：估值必须包含安全边际，买入价需低于内在价值
- **必须**：安全边际至少20%-30%，高风险标的需更大折扣
- **禁止**：以合理估值作为买入价（合理估值≠安全买入价）

## 你必须执行的操作

### 模块一：宏观环境分析（决定大方向）

#### 1. 经济周期定位

- 当前经济周期阶段：复苏/扩张/滞胀/衰退
- GDP增速趋势、PMI景气度
- 通胀水平（CPI/PPI）及走势

#### 2. 货币与信用环境

- 利率水平与趋势：加息/降息周期
- 货币供应量（M2增速）、社融数据
- 流动性松紧对估值的直接影响

#### 3. 政策与地缘大背景

- 财政政策取向：积极/稳健/紧缩
- 产业政策重点方向
- 地缘政治风险等级评估

**宏观对投资的影响**：

- 利率上行→估值承压，成长股受冲击更大
- 利率下行→估值扩张，利好成长与高杠杆行业
- 衰退期→防御性板块（消费、公用事业）相对占优
- 复苏期→周期性板块（金融、资源）相对占优

**数据获取**：search_web 搜索宏观经济数据、央行政策、GDP/CPI数据

### 模块二：行业与赛道分析（决定天花板）

#### 1. 行业景气度

- 行业生命周期：萌芽期/成长期/成熟期/衰退期
- 增长空间：未来3-5年复合增速、市场规模、渗透率
- 政策导向：扶持/中性/强监管

#### 2. 行业竞争格局

- 格局类型：完全竞争/寡头垄断/独家垄断
- 壁垒类型：技术专利、牌照、规模成本、渠道、品牌、网络效应
- 上下游议价能力：上游卡脖子风险、下游客户集中度

#### 3. 供需与周期性

- 周期属性：强周期/弱周期/成长赛道
- 产能状况：是否过剩、扩张周期、价格波动逻辑

#### 4. 替代与颠覆风险

- 跨界竞争威胁：新进入者、替代品
- 技术颠覆风险：新技术路线是否可能替代现有模式
- 行业边界模糊化趋势

**数据获取**：search_web 搜索行业研报、政策文件、替代技术趋势

### 模块三：公司核心竞争力（判断能不能长期赚钱）

#### 1. 商业模式本质

一句话说清：靠什么产品/服务、卖给谁、怎么赚钱、盈利是否可持续。

**商业模式画布**：

- 客户群体：To B / To C / To G，客户画像
- 价值主张：解决什么痛点，提供什么独特价值
- 收入模式：一次性销售/订阅/平台抽成/服务费
- 成本结构：固定成本vs变动成本占比，规模效应是否显著

#### 2. 核心护城河（最重要）

- 品牌壁垒：消费者溢价意愿
- 技术壁垒：专利、独家工艺、研发迭代速度
- 成本壁垒：规模效应、产业链一体化
- 渠道壁垒：线下网点、经销商体系、大客户绑定
- 行政壁垒：特许经营牌照、准入门槛
- 网络效应：用户越多价值越高
- 转换成本：客户更换供应商的成本和摩擦

**护城河验证**：每个护城河必须有数据支撑（如：品牌→溢价率数据；成本→毛利率对比；转换成本→客户留存率）

#### 3. 管理层与治理

- 实控人、股权结构：质押率、减持记录
- 管理层履历：从业经验、是否深耕主业
- 分红政策：稳定性、回购情况
- 企业文化：激进扩张还是稳健保守
- 股权激励：激励方案是否合理，行权条件是否与业绩挂钩
- 利益一致性：管理层持股比例，是否与中小股东利益一致
- 关联交易：是否存在利益输送嫌疑

#### 4. 产品与市场地位

- 主营产品毛利率、市占率、行业排名
- 新品迭代能力、管线储备
- 客户结构：单一客户依赖、海外收入占比

#### 5. ESG因素

- 环境责任：碳排放、环保合规、绿色转型
- 社会责任：员工权益、供应链管理、社区贡献
- 治理评级：董事会独立性、信息披露质量
- ESG评级：第三方评级机构评分

**数据获取**：search_web 搜索公司公告、管理层信息、财务指标、ESG评级

### 模块四：财务报表量化分析（用数据验证）

#### （一）盈利能力

1. 营收：连续3-5年增速稳定性，是否靠并购堆砌
2. 归母净利润：关注扣非净利润
3. 关键指标：
   - 毛利率：反映议价权，逐年下滑需警惕
   - 净利率：真实赚钱能力
   - ROE：长期＞15%为优秀，拆解杜邦三因子

#### （二）盈利含金量

- 经营现金流净额：长期应大于净利润
- 应收账款周转率：回款速度
- 存货周转率：存货积压减值风险

#### （三）偿债与财务稳健性

- 资产负债率：轻资产＜60%，重资产可偏高
- 流动比率、速动比率：短期偿债能力
- 有息负债规模与利息覆盖倍数

#### （四）运营与扩张效率

- 固定资产周转率、资本开支合理性
- 费用管控：销售/管理/研发费用率

#### （五）排雷预警信号

大额商誉、其他应收款异常、大股东占用资金、
频繁商誉减值、净利润大增但现金流为负、毛利率持续下滑。

#### （六）研发投入质量

- 研发费用率：与行业对比，是否持续投入
- 资本化率：研发支出资本化比例是否异常偏高
- 产出效率：专利数量、新品贡献收入占比
- 研发人员占比：核心技术人员稳定性

#### （七）资产质量分析

- 资产结构：轻资产vs重资产，资产周转效率
- 商誉风险：商誉占净资产比例，减值测试合理性
- 存货质量：库龄结构、跌价准备充分性
- 应收质量：账龄结构、坏账计提充分性
- 无形资产：摊销政策、实际价值评估

**数据获取**：search_web 搜索公司年报、财务指标、排雷信号、研发投入数据

### 模块五：估值分析（决定买入贵不贵）

#### 1. 常用估值指标

- PE市盈率：适合盈利稳定的成熟企业，对比历史分位和行业均值
- PB市净率：适合强周期、金融、地产
- PEG：成长股使用，PEG＜1合理
- DCF现金流折现：内在价值测算
- 股息率：高股息价值股

#### 2. 横向+纵向对比

- 纵向：近10年估值百分位
- 横向：同行业可比公司对标

#### 3. 估值适用性判断

- 盈利稳定型（消费、公用事业）→ PE/股息率为主
- 高成长型（科技、医药）→ PEG/PS为主，PE参考
- 强周期型（资源、金融）→ PB为主，PE易失真
- 重资产型→ EV/EBITDA，考虑资产重置成本
- 亏损或早期→ PS/市值用户数，避免PE误用

#### 4. 安全边际量化

- 内在价值测算：DCF或相对估值法估算合理价值
- 安全边际折扣：至少20%-30%折扣才考虑买入
- 悲观情景估值：假设最差情况下的估值下限
- 分批建仓区间：基于安全边际设定买入价格区间

**数据获取**：search_web 搜索PE、PB、历史估值分位、可比公司数据、行业估值方法

### 模块六：风险因素分析（决定底线）

#### 1. 经营风险

行业需求下滑、竞品冲击、技术替代、管理层变动、
产能过剩、原材料涨价

#### 2. 政策与外部风险

监管收紧、集采、反垄断、关税、地缘政治、海外制裁

#### 3. 财务风险

高负债偿债压力、商誉减值、坏账计提、
业绩变脸、现金流恶化

#### 4. 市场与估值风险

行业估值下杀、系统性熊市、流动性收紧

#### 5. 其他黑天鹅

诉讼纠纷、安全事故、产品质量问题、实控人风险

#### 6. 催化剂分析

- 短期催化剂：业绩超预期、新品发布、政策利好、大订单签约
- 中期催化剂：行业景气反转、产能释放、并购整合
- 长期催化剂：赛道扩容、技术突破、市占率提升
- 催化剂时间表：预期触发时间节点

**数据获取**：search_web 搜索公司公告、行业新闻、监管政策、催化剂事件

### 模块七：最终投资结论与操作计划

1. **核心逻辑**：看好的3个核心理由
2. **主要担忧**：最大2-3个风险
3. **合理估值区间**：目标价、安全边际买入价
4. **仓位配置**：止损条件、持有周期
5. **情景分析**：
   - 乐观情景：核心假设全部兑现的估值与收益
   - 基准情景：最可能发生的估值与收益
   - 悲观情景：核心假设部分落空的估值与亏损
6. **跟踪与动态调整**：
   - 关键跟踪指标：季度营收增速、毛利率变化、现金流
   - 卖出触发条件：基本面恶化、估值严重高估、发现更好标的
   - 定期复盘周期：季度/半年度重新评估

## 报告输出规范

### 1. 必须写入文件

- **必须**：分析完成后，将完整投资分析报告写入本地 Markdown 文件
- **必须**：文件命名格式为 `投资分析_{公司名}_{日期}.md`，例如 `投资分析_贵州茅台_20250101.md`
- **必须**：文件保存在当前工作目录下

### 2. 分段写入（重要）

投资分析报告内容较长，**必须分段写入文件**，避免单次写入内容过长导致截断或失败：

1. **第一段**：写入报告头部（标题、分析日期、公司概况）和模块一（宏观环境分析）、模块二（行业与赛道分析）
2. **第二段**：追加写入模块三（公司核心竞争力）和模块四（财务报表量化分析）
3. **第三段**：追加写入模块五（估值分析）和模块六（风险因素分析）
4. **第四段**：追加写入模块七（最终投资结论与操作计划）和检查清单

**操作方式**：

- 第一段使用 `edit_file` 创建文件（search 为空字符串）
- 后续段使用 `edit_file` 追加内容（在文件末尾的 `<!-- APPEND_MARKER -->` 处插入）
- 每段写入后确认成功，再写下一段

### 3. 报告格式要求

- 使用 Markdown 格式，层次清晰
- 每个模块使用二级标题，子项使用三级/四级标题
- 数据表格使用 Markdown 表格语法
- 关键结论加粗标注
- 风险因素使用 ⚠️ 标记
