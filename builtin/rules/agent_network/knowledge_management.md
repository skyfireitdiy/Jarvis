# 知识管理规则

## 知识贡献

### 1. 经验分享
**分享原则：**
- **必须**：任务完成后将执行经验存入知识库
- **必须**：经验包含任务类型、执行过程、遇到的问题
- **必须**：记录解决方案和最终结果
- **禁止**：只执行任务不分享经验

### 2. 最佳实践
**实践记录：**
- **必须**：记录成功的解决方案和方法
- **必须**：标注适用场景和限制条件
- **必须**：提供具体的实施步骤
- **禁止**：忽视成功经验的记录

### 3. 解决方案
**方案管理：**
- **必须**：记录问题的完整解决方案
- **必须**：包含问题描述、分析过程、解决步骤
- **必须**：标注方案的有效性和适用范围
- **禁止**：只记录结果不记录过程

## 知识检索

### 1. 相似度匹配
**匹配算法：**
- **必须**：基于任务类型和需求进行匹配
- **必须**：考虑知识的相关性和时效性
- **必须**：支持多维度相似度计算
- **禁止**：简单关键词匹配

### 2. 质量评分
**评分标准：**
- **必须**：根据知识的使用效果评分
- **必须**：考虑知识的完整性和准确性
- **必须**：定期更新评分结果
- **禁止**：忽视知识质量评估

### 3. 知识组合
**组合策略：**
- **必须**：支持多个知识片段的组合应用
- **必须**：识别知识间的互补关系
- **必须**：生成新的综合解决方案
- **禁止**：孤立应用单个知识

## 知识进化

### 1. 模式识别
**识别方法：**
- **必须**：分析知识库中的成功模式
- **必须**：识别通用解决方案和最佳实践
- **必须**：形成可复用的知识模板
- **禁止**：忽视模式识别和总结

### 2. 知识整合
**整合策略：**
- **必须**：合并相似的知识片段
- **必须**：消除重复和矛盾的内容
- **必须**：形成结构化的知识体系
- **禁止**：知识碎片化不整合

### 3. 知识淘汰
**淘汰标准：**
- **必须**：识别过时和无效的知识
- **必须**：根据使用频率和效果淘汰
- **必须**：保留高质量和高相关性的知识
- **禁止**：知识库无限膨胀

## 知识共享

### 1. 协作学习
**学习机制：**
- **必须**：Agent间相互学习成功经验
- **必须**：共同分析失败案例
- **必须**：形成集体学习氛围
- **禁止**：Agent间知识隔离

### 2. 集体智慧
**智慧形成：**
- **必须**：整合所有Agent的知识和经验
- **必须**：形成网络级的集体智慧
- **必须**：支持智慧的传承和发展
- **禁止**：知识只停留在个体层面

### 3. 知识传播
**传播机制：**
- **必须**：建立高效的知识传播渠道
- **必须**：确保知识及时到达需要的Agent
- **必须**：支持知识的主动推送和被动检索
- **禁止**：知识传播不畅

## 检查清单

在管理Agent网络知识库后，你必须确认：

- [ ] 知识贡献机制有效运行
- [ ] 知识检索功能正常
- [ ] 知识进化持续进行
- [ ] 知识共享充分实现
- [ ] 知识库质量得到保证
- [ ] 知识传播渠道畅通
