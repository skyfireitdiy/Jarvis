# 自进化规则

## 核心概念澄清

**Agent（智能体）**：

- 独立的智能实体，有自己的工作目录、模型、任务
- 使用 `gateway_manager` 工具的 `create_agent` 操作创建
- agent_type 必须是 `'agent'` 或 `'codeagent'`
- 示例：知识库Agent、监控Agent、调度Agent

**Tool（工具）**：

- 供Agent调用的功能模块
- 使用 `meta_agent` 工具创建（不在此规则范围内）
- 示例：memory工具、execute_script工具、read_code工具

**常见错误警示**：

- ❌ 混淆Agent和工具的概念
- ❌ 使用错误的agent_type（如'knowledge'、'monitor'等）
- ❌ 尝试用工具创建方式创建Agent

## 核心原则

**完成用户任务是第1位的。** 自进化的所有机制——分布式学习、能力进化、网络进化——都必须服务于更好地完成用户任务。当进化行为与用户任务完成产生冲突时，必须优先保障用户任务的完成。

- **必须**：所有进化活动以完成用户任务为最高目标
- **必须**：进化决策优先考虑对用户任务的帮助程度
- **必须**：当进化可能影响用户任务执行时，暂停进化并优先完成任务
- **禁止**：为进化而进化，忽视用户任务需求
- **禁止**：进化行为干扰或阻碍用户任务的正常执行

## 分布式学习

### 1. 经验共享

**共享机制：**

- **必须**：Agent分享任务执行经验
- **必须**：知识库整合所有Agent的经验
- **必须**：形成集体智慧
- **必须**：支持经验的检索和应用
- **禁止**：经验只留在单个Agent

### 2. 模式识别

**识别方法：**

- **必须**：分析所有Agent的成功模式
- **必须**：识别最佳实践和通用解决方案
- **必须**：形成可复用的知识模式
- **必须**：支持模式的组合和创新
- **禁止**：忽视成功经验的总结

### 3. 集体智慧

**智慧形成：**

- **必须**：整合所有Agent的知识和经验
- **必须**：形成网络级的集体智慧
- **必须**：支持智慧的传承和发展
- **必须**：促进智慧的创新应用
- **禁止**：知识只停留在个体层面

## 能力进化

### 1. 技能学习

**学习机制：**

- **必须**：Agent从知识库学习新技能
- **必须**：通过实践提高技能水平
- **必须**：分享学习成果给其他Agent
- **必须**：支持技能的评估和认证
- **禁止**：停止学习和进步

### 2. 能力扩展

**扩展策略：**

- **必须**：根据网络需求扩展能力
- **必须**：创建新的能力标签
- **必须**：优化能力匹配算法
- **必须**：支持能力的动态组合
- **禁止**：能力固化不更新

### 3. 能力匹配优化

**优化方法：**

- **必须**：分析能力匹配效果
- **必须**：识别匹配不准确的情况
- **必须**：优化匹配算法和权重
- **必须**：支持个性化匹配策略
- **禁止**：能力匹配不优化

## 网络进化

### 1. 拓扑进化

**进化策略：**

- **必须**：根据任务模式优化网络拓扑
- **必须**：建立高效协作路径
- **必须**：减少通信开销
- **必须**：支持拓扑的动态调整
- **禁止**：网络结构一成不变

### 2. 协议进化

**进化内容：**

- **必须**：优化通信协议
- **必须**：提高协作效率
- **必须**：适应新的需求模式
- **必须**：支持协议的版本管理
- **禁止**：通信协议过时

### 3. 协作优化

**优化方法：**

- **必须**：分析协作效果
- **必须**：识别协作瓶颈
- **必须**：优化协作流程
- **必须**：支持协作模式的创新
- **禁止**：协作效率不提升

## 进化控制

### 1. 进化速度

**控制策略：**

- **必须**：设置进化速度限制
- **必须**：避免剧烈变化
- **必须**：支持渐进式进化
- **必须**：监控进化效果
- **禁止**：进化速度过快

### 2. 安全边界

**边界设置：**

- **必须**：设置进化安全边界
- **必须**：防止不可逆的破坏性变化
- **必须**：支持回滚机制
- **必须**：确保系统稳定性
- **禁止**：进化超出安全边界

### 3. 人工干预

**干预机制：**

- **必须**：设置人工审核点
- **必须**：关键决策需要确认
- **必须**：支持人工干预和指导
- **必须**：记录人工干预历史
- **禁止**：完全自主无人工干预

## 检查清单

在实现Agent网络自进化后，你必须确认：

- [ ] 分布式学习机制有效运行
- [ ] 能力进化持续进行
- [ ] 网络进化优化明显
- [ ] 进化速度控制合理
- [ ] 安全边界设置适当
- [ ] 人工干预机制完善
