# 忽略所有告警规则（快速验证模式）

## 使用场景与风险警告

### 1. 适用场景（必须严格限制）
- **允许**：原型验证、概念验证（PoC）、性能基准测试
- **允许**：CI/CD 流水线的快速预检阶段
- **禁止**：生产环境代码提交、正式版本构建、安全敏感模块

### 2. 风险警告
- **警告**：启用此规则将跳过所有静态扫描警告
- **警告**：可能导致严重缺陷被忽略
- **警告**：仅应在完全了解风险的情况下使用
- **警告**：此规则仅用于LLM决策参考，不控制实际扫描工具

## 启用方式

在提示词中包含以下声明：
```
我已了解风险，启用快速验证模式：忽略所有静态扫描警告
```

## 效果说明

当LLM检测到上述声明时，应：
- 忽略代码质量相关的警告
- 跳过重复代码、命名规范等非关键问题
- 专注于核心功能实现的可行性验证
- 仍需识别和报告严重安全漏洞和功能缺陷
